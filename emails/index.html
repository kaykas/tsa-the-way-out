Design Notes

  • Fonts: Trajan Pro 3 (headings) + Proxima Nova (body) via Adobe Typekit — same kit as the microsite. Falls back to Georgia / Helvetica Neue in email clients.
  • Colors: TSA Red #ED1C2B for CTAs and accents · Navy #2D3E63 for header/footer · Body text #111111
  • Layout: Table-based HTML, 600px max-width, inline CSS. Tested for Gmail / Outlook / Apple Mail. No rounded corners.
  • CTAs: Emails 1–2 drive to the program page. Emails 3–5 drive to the donation page.
  • Stats: Leading with 77% success rate (program exit) — consistent with positioning framework.
